What is LifeSaver?

LifeSaver is a demonstration of how technology can help us approach ongoing social crises. Our project is intended to serve as a local alert network for situations involving an active shooter. The site allows users to sign up with email and phone, report emergencies with text/audio/location information, and alert registered users within a 10 mile radius.

How was LifeSaver created?

To help explain how we created LifeSaver, we created an outline of our project that is linked here.
Some of the technology/APIs included in LifeSaver include:
  • Google Cloud Datastore
  • Google Cloud Firebase
  • Gmail API
  • MediaRecorder API
  • Google Geocoding API
  • ZIP Code API
  • HTML5 Geolocation
